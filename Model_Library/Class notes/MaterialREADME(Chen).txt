A class to store material information
Made by Chen Xu

The class allows to store ID, density, color and name of a material of a VTK model.
Information can be set or retrieved.

Parameters of information:
ID - an integer identifying the material among other materils in the model'
Density - decribes material density
Color - a Vector3D object describing RGB values of the color of the material ( range: 0.0-1.0).
Name - name of the material.
